When dealing with problems, first we need to understand what we need to do and what problem we want to solve.

Algorithms are nothing more than a logic order of actions that need to be done, in order to solve a given problem.

## Flowchart

A flowchart is a visual representation of a logical  command sequence we usually have in our minds. 

Assume we need to create a shelter for our character in a game. So the monsters don't kil him. For the steps, we will
draw a flowchart. This flowchart will also be on the same folder.
